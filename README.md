#  4.建议
1.一定要有**独立思考**的能力，不能一看题目觉得不会立马就看题解，说实话这样做单纯就是浪费时间和这些好题目。

2.如果**实在没思路**或者说已经**卡了三四个小时**了，那就建议看看题解，否则自己的游戏体验也会很差，但是看完题解还是要**自己尝试写代码**，否则Ctrl c+v也是没有什么实际意义的。

3.剑指offer本身的题难度还是不错的，既有水题，也有一些中高档的题，还是比较合适刷的，并且剑指offer也是很多公司经典的题目，做一做还是不错的。

4.刷题的平台我用的是**牛客网**，但是自己刷下来以后发现其实牛客的很多样**例设计的其实不是非常的规范**，像有些题，我自己A过以后其实回头去看看别人的题解会发现自己的思路其实是不完善的，但是牛客网还是过了。所以说如果**只是玩玩建议牛客**，否则**最好还是去LeetCode上面去刷**，那里的样例应该会健壮很多。

5.剑指offer上的大部分题目都是**跟数据结构有关**，比如**链表，二叉树，栈**等等，所以建议大家还是最好是有数据结构这方面的基础再来刷，否则可能效率会很低，否则游戏体验会很差。

6.碰到二叉树，链表等题的时候建议大家最好先返回一下他给的节点，看看是什么格式的，因为每个平台的样例都不同，所以最好是先**适应每个平台的样例格式**，否则因为这个可能也会卡很长的时间。

7.牛客刷题的时候会有**练习与考试两种模式**，大家**平常练习就用练习模式**，然后选择自己的编译器进行编写就好了，毕竟在线编辑器那玩意儿是真的不好用，但是如果是**求职的，可以选择考试模式**。并且就用在线编辑器，可以更好的帮助自己适应公司机试。
# 5. 感想
1.   **你知道的越多，你不知道的就越多。**   我真的很欣赏这句话，我们学技术的真的需要谦虚，毕竟山外有山，人外有人。做题过程中，碰到一道我做出来的题，觉得这一定是最优解时，看了看别人的题解，才知道人家那个才是真的牛逼，我的纯属闹着玩。

2. **尽量保持一颗平常心**，出现bug并不可怕，重点是解决bug，最后两天博主的心态爆炸的狠，主要是题目真的刚不出来，有点拖节奏。其实还是要改改的，毕竟题目是死的，人是活的。实在刚不过就看看题解呗。
